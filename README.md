# 九江农产品专销平台微信小程序

## 前言

本项目是基于微信小程序的九江农产品专销平台，旨在为广大农户和消费者提供一个便捷、高效的农产品交易平台。项目采用Java语言，结合Spring Boot框架，前端采用JS、Vue和CSS3等技术，数据库使用MySQL 5.7/8.0，开发工具为IDEA或Eclipse。以下为项目详细介绍。

## 内容介绍

本项目主要包括以下模块：用户模块、商品模块、订单模块、支付模块等。用户模块负责用户注册、登录、修改个人信息等功能；商品模块负责商品的上架、下架、编辑和展示；订单模块负责订单的生成、支付、取消等操作；支付模块负责处理支付请求和通知。通过这些模块的协同工作，实现了农产品专销平台的基本功能。

## 技术介绍

- 语言：Java
- 使用框架：Spring Boot
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16

## 核心代码

以下是项目中的一个核心代码示例，展示了商品模块中查询商品列表的方法：

```java
// 商品服务类
@Service
public class ProductService {

    // 注入商品DAO
    @Autowired
    private ProductDao productDao;

    // 查询商品列表
    public List<Product> listProducts() {
        return productDao.listProducts();
    }
}
```
